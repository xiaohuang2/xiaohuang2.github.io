Aplayer 与 Metingjs 简介

Aplayer 是一个很棒的用于 HTML5 的音乐播放器，官方文档可以在 这里 看。第一句话就是 “🍭 Wow, such a beautiful HTML5 music player.”

pjax 简介

pjax 即为 pushState + ajax ，可实现预加载并刷新指定的局部页面。也就是说，我们可以通过 pjax 来实现刷新网页而不刷新音乐播放器，以达到让音乐变为全局的 bgm 那样的效果。

而那句 script 的意思就是告诉 pjax 只需对给了 id “container” 的内容进行处理。

这也是为什么那段 aplayer 的代码的 div 标签也给了一个 id 叫 my-aplayer 。就是为了区分。
