Markdown 基础语法

众所周知，Markdown是一个专用于写博客的语言，Markdown文件后缀为.md。它的语法很简单，本篇文章就结合网上的资料向大家介绍一下。

标题

标题分六级，分别用对应数量的#字标识。

一级标题如上所示 ↑

输入：

## 二级标题

### 三级标题

#### 四级标题

以上输入分别可以输出二级、三级、四级标题。

五级六级以此类推。注意，#字后有一个空格。如果你想在你的md文件中直接展示语法（如上文我写的“输入”），只需在#号前添加反斜杠‘\’即可，如“\#”。

段落

段落在Markdown中很简单，只是要注意不要在段落开始空格或缩进（除非在列表中，列表后面会讲），段落与段落中间有空行。

输入：

This is the first paragraph.

（空行）

This is the second paragraph.

输出：

This is the first paragraph.

This is the second paragraph.

不要有以下输入：

​ This is the first paragraph. And this is a

sentence.

​ This is the second paragraph. And this is also a

sentence.

换行

语法是先在一行末尾打两个空格，再按下回车。

输入：

First line.
Second line.

输出：

First line.
Second line

粗体与斜体

粗体只需在文本两侧分别添加上没有空格的**或__（两道下划线）即可，斜体只需一个*或_即可。

输入：

**Bold text** __Bold text__

*Italic text* _Italic text_

***Important text*\ **_Important text_**

输出：

Bold text Bold text

Italic text _Italic text_

Important text _Important text_

只要符合粗体或斜体语法，任意组合均可。

区块引用

区块引用前文用到过，输入输出都用的这个。只需‘>’符号即可，但‘>’后有一个空格。

输入：

> haha

> hahaha

> hahahaha

输出：

haha

hahaha

hahahaha

还有嵌套用法：

输入：

> haha

>> hahaha

输出：

haha

hahaha

如果你担心兼容性，可在区块前后放置空行。

列表

无序列表

无序列表只需在每行前放置‘+’或‘-’或‘*’即可。

输入：

+ First item

+ Second item

​ Another paragraph（只需在段落开头缩进）

+ Third item

输出：

  • First item

  • Second item

    ​ Another paragraph

  • Third item

有序列表

有序列表只需两个条件：

  1. 每行由数字和句点开头
  2. 第一行数字为1

上面就是一个有序列表。

所以这样的输入：

1. ha
3. hahaha
2. haha

也能输出一个有序列表：

  1. ha
  2. hahaha
  3. haha

CodeBlock（代码块）

我最爱的代码块终于来了。

谈代码块之前，先说一下代码，只需用反单引号括起来即可（注意输入法要调英语才有反单引号，如这样的：`）

输入：`hexo -v`

输出：hexo -v

代码块只需用三个反单引号（```）或三个波浪线（~~~）括起来即可。

如输入：

```python（这里指定语言，用于显示语法高亮）
print(“Hello world.”)

```

链接

想插入链接，只需先用[]将需要显示的文本括起来，再用()把文本去到的链接括起来即可。还可在url后面用双引号括上一段文字，作为鼠标悬浮在文本上时的提示文本。

输入：

[不懂的问度娘](https://baidu.com \”这儿是百度，不懂就去搜啊\”)

输出：

不懂的问度娘

如果你只是单纯想把URLs或E-mail Addresses转化成链接，请用尖括号将其括起来。

如输入：

\xiaohuang21314@gmail.com\

输出：

xiaohuang21314@gmail.com

之前说到的文本格式符都能运用到链接上，如输入：

**[不懂的问度娘](https://baidu.com \”这儿是百度，不懂就去搜啊\”)**

则输出：

不懂的问度娘

图片

首先输入一个叹号“！”然后用中括号[]将加载时要显示的文本括起来，最后用小括号()将图片地址与鼠标悬浮在图片上时显示的文本括起来。

输入：![Rem is cute](https://s2.loli.net/2022/01/29/ChRAmP2aNIUSnLd.png “Is Rem cute? Do you like her?”)

输出：还可以给图片添加链接，即再用一对中括号把图片链接括起来，然后在后面加小括号，小括号中填上链接即可。

输入：[!Rem is cute\](https://zh.wikipedia.org/wiki/%E9%9B%B7%E5%A7%86_(%E8%A7%92%E8%89%B2))

输出：)

点击图片即可去到wiki百科。

结束语 & 总结

so，以上就是Markdown的基础语法讲解。想学更多可以去Markdown Guide了解。
