C 语言中 flag 的运用

C 语言中有一种变量叫 flag（标记），今天来看两个实例，均来自《C Primer Plus》。

Program list 7.5—-divisors.c

这是程序清单 7.5 的程序，用于判断一个数是否为素数，如果是，则报告，如果不是，则显示该书的所有约数。

我并没有完全按照书上来写，有些地方可改进。

在该程序中， is_prime 即为一个标记，使用了 _Bool 类型的变量。从第 12 行开始时， for 循环首先将 is_prime 初始化为 true ，如果 num （待测试数）能够进入到内层的 if 语句，说明该数是有约数的，则将 is_prime 设为 false 。当然，没有进入 if 的数，自然就是素数了， is_prime 仍为 true 。
